## Exercise: Temperature Converter
### Goal: 
Practice conditional statements and basic arithmetic by converting between Celsius and Fahrenheit.

### Instructions
This exercise re-iterates everything you've learned about conditionals and loops to convert temperatures between Fahrenheit and Celsius. First, take the input from the user on which conversion they want to do, and then convert based on that conditional.

Ask the user which conversion they want:
Celsius to Fahrenheit
Fahrenheit to Celsius
Read the user’s choice and print the appropriate answer
Otherwise, print "Invalid choice!".
Bonus Challenge (Optional) Add a loop so that after converting one temperature, the program asks the user if they want to convert another.

(Use a while loop with a char convertAgain = 'y'check.)
